# Research

Languages: [English](README.md) · **Русский**

Заметки по поднятию платы EBAZ4205 (Zynq-7010) и эксперименты — каждый шаг самодостаточен: исходники и готовый к прошивке bitstream. Это кирпичи, на которых стоят ядра BulbuLator, записанные по мере проверки на реальном железе.

| Шаг | Что охватывает |
|-----|----------------|
| [00 — Настройка и разводка](00-setup/) | С нуля: питание, перемычка режима загрузки, JTAG-программатор (кабель или Pico), установка ПО и прошивка bitstream-а |
| [01 — Поднятие платы: мигание светодиода](01-board-bringup-blink/) | Питание, путь JTAG → PL и работающий bitstream только на внутреннем осцилляторе чипа |
| [02 — Кнопки управляют светодиодами](02-buttons-and-leds/) | Чтение четырёх кнопок шилда (инвертированные, синхронизированные) и управление ими светодиодами |
| [03 — Примитивный HDMI: цветные полосы](03-hdmi-bars/) | Первая картинка на экране — восемь цветных полос 720p, тактируемых от PS (урок про уровневые сдвигатели / `ps7_post_config`) |
| [04 — HDMI с переключаемыми паттернами](04-hdmi-buttons/) | Прыгающий квадрат и четыре паттерна (полосы, градиент, шахматная доска), переключаемые кнопками шилда |
| [05 — HDMI аудио: квадрат пищит](05-hdmi-beep/) | Первый звук — прыгающий квадрат бипает по HDMI audio при ударе о стену, через ядро hdl-util/hdmi |
| [06 — ZX Spectrum 128](06-zx-spectrum-128/) | Всё сходится: настоящий ZX Spectrum 128 на плате — HDMI видео + аудио, кнопки шилда управляют загрузочным меню, загрузка игр с ленты через пин и загрузка с SD. Собрано на ядре Atlas `zx` |
| [07 — Будим ARM: PS↔PL control plane](07-arm-control-plane/) | AXI-интерфейс регистров, через который простаивавший ARM умеет останавливать Z80 и читать/писать память Спектрума вживую — фундамент для загрузки игр с SD. Чистое надмножество Шага 6 |
| [08 — Картинка без разрывов: DDR framebuffer с двойной буферизацией](08-ddr-framebuffer/) | Буферирует 51 КБ ZX-кадра в PS DDR (тройной буфер, переключение по HDMI vblank), чтобы демо с эффектами бордюра и переключения теневого экрана перестали рваться — первое реальное использование AXI-HP пути к DDR |

Шаги 0–5 — взлётная полоса; Шаг 6 — первая настоящая машина. Дальше заметки по поднятию платы переходят в портирование ядер (больше точности, больше демо и в итоге более сложные машины, которым нужен PS DDR).
